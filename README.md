# ERC-1450 Reference Implementation

Official reference implementation of the ERC-1450 RTA-Controlled Security Token Standard.

## Overview

This repository contains the reference implementation for [ERC-1450](https://github.com/StartEngine/ERCs/blob/update-erc-1450-revival/ERCS/erc-1450.md), a token standard designed for compliant securities offerings under SEC regulations (Reg CF, Reg D, Reg A).

## Prerequisites

- Node.js v16+ (tested with v22.14.0)
- npm v7+ (tested with v10.9.2)

## Quick Start

```bash
# Clone the repository
git clone https://github.com/StartEngine/erc1450-reference.git
cd erc1450-reference

# Install dependencies
npm install

# Compile contracts
npm run compile

# Run tests
npm test
```
